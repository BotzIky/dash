qbot example code

qbot adalah chat bot menggunakan database

Informasi Penting

qbot adalah chat bot yang menggunakan database sebagai kekuatan dalam berinteraksi dengan pengguna, contoh bot yang menggunakan qbot bisa di coba melalui bot berikut @ValRisk dengan perintah:
/qbot

Kalian juga bisa membantu mengembangkan database jika ingin berkontribusi dalam pengembangan qbot ini, setiap database yang dibuat akan membuat qbot semakin responsif, terima kasih untuk kalian yang membantu mengembangkan database, berikut cara menambahkan database qbot melalui bot @Git_Keybot dengan perintah:
/qapi (pertanyaan|jawaban)
Contoh: /qapi kamu siapa?|aku adalah qbot

Code qbot

Simple code yang dapat di implementasikan dalam projek javascript: 


Create by IG: @iikyynd_04

const text = 'PERTANYAAN' // ubah PERTANYAAN menjadi pertanyaan untuk di ajukan ke qbot
    const fs = require('fs'); // module package.json "fs": "^0.0.1-security" 
    const axios = require('axios'); // module packakge.json "axios": "^1.6.7"
    try {
    const loadDatabase = async () => {
    const url = 'https://api.riskynd.eu.org/question/database.json'; // don't replace it!
    const response = await axios.get(url);
    return response.data;
     };
    const levenshteinDistance = (a, b) => {
    const matrix = [];
    for (let i = 0; i <= b.length; i++) {
        matrix[i] = [i];
    }
    for (let j = 0; j <= a.length; j++) {
        matrix[0][j] = j;
    }
    for (let i = 1; i <= b.length; i++) {
        for (let j = 1; j <= a.length; j++) {
            if (b.charAt(i - 1) === a.charAt(j - 1)) {
                matrix[i][j] = matrix[i - 1][j - 1];
            } else {
                matrix[i][j] = Math.min(
                    matrix[i - 1][j - 1] + 1,
                    Math.min(
                        matrix[i][j - 1] + 1,
                        matrix[i - 1][j] + 1
                    )
                );
            }
        }
    }
    return matrix[b.length][a.length];
     };
    const fetchAnswer = async (question) => {
    const db = await loadDatabase();
    let closestMatch = null;
    let closestDistance = Infinity;
    db.data.forEach(item => {
        const distance = levenshteinDistance(item.question.toLowerCase(), question.toLowerCase());
        if (distance < closestDistance) {
            closestDistance = distance;
            closestMatch = item;
        }
    });
    return closestMatch ? closestMatch.answer : "Maaf, aku nggak ngerti";
     };
    const answer = await fetchAnswer(text);
    console.log(answer); ganti dengan sistem pengiriman pesan
    } catch (error) {
        console.log(error);
    }

qbot example code

qbot is a chat bot using a database

Important Information

qbot is a chat bot that uses a database as a power to interact with users. Examples of bots that use qbot can be tried using the following bot @ValRisk with command:
/qbot

You can also help develop the database if you want to contribute to the development of this qbot, every database created will make the qbot more responsive, thank you to those of you who helped develop the database, here's how to add a qbot database via bot @Git_Keybot with command:
/qapi (question|answer)
Contoh: /qapi who are you?|I am qbot
